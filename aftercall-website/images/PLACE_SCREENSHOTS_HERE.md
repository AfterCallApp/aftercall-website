# AfterCall Website — Screenshot Setup

Save your screenshots to this `images/` folder using the filenames below.
The website will automatically display them in the correct sections.

## Where Each Screenshot Appears

- **Hero phone** → `dashboard.png`
- **"Log Every Call" feature card** → `new-call.png`
- **"Know Your Shifts" feature card** → `my-calls-analytics.png`
- **"Process What You Carry" feature card** → `journal-checkin.png`
- **"Connect With Your Crew" feature card** → `community-feed.png`
- **Dark Mode Showcase section** → `dashboard-dark.png` (large phone) + `shift-calendar.png` (offset phone)
- **Scrolling gallery** → all 22 screenshots (15 light + 7 dark mode, interspersed)

## Tips

- Screenshots can be any resolution — the site crops them automatically (top-aligned)
- PNG or JPG both work fine
- Don't rename the files or the website won't find them
